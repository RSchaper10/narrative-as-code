# Changelog

All notable changes to this project should be documented here.

The format is inspired by Keep a Changelog, and this project follows a lightweight version of Semantic Versioning for public releases.

## [Unreleased]

- Added public-repo governance and contribution docs.
- Added issue templates and a pull request template.
- Added a project bootstrap script for turning the sample into a fresh manuscript workspace.
- Added dependency installation guidance for common local setups.
- Added launch art prompt guidance for DALL-E-style promotional artwork.

## [0.1.0] - 2026-03-29

- Initial public starter release.
- Added local-first manuscript structure with sample project files.
- Added schema-backed validation for project, chapter, and audiobook metadata.
- Added build pipeline for compiled manuscript, stats, continuity, and project reports.
- Added import utility for monolithic Markdown drafts.
- Added playbooks for review, refinement, continuity, metadata updates, and editor packet prep.
- Added CI workflow and automated tests for core scripts.
