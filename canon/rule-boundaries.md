# Rule Boundaries

These are working separations that keep the mystery coherent.

## Signal

- The signal is detectable through instruments before characters understand it.
- It may affect behavior or perception, but it should not function like instant mind control.
- The narrative should not confirm its origin too early.

## Infrastructure

- Old siren hardware is part of the phenomenon-management system, not decorative set dressing.
- Modern maintenance patches imply current use.
- Institutional concealment should look bureaucratic first and sinister second.
