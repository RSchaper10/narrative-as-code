# Demo Walkthrough

Use this sequence for a tight 3-5 minute demo of the starter.

## Arc

1. Show the repo map in the root README.
2. Open one chapter, one canon file, and one metadata file to show source-of-truth separation.
3. Run `python3 scripts/check-setup.py`.
4. Run `python3 scripts/validate-project.py`.
5. Run `./scripts/build-manuscript.sh`.
6. Open:
   - `build/manuscript-stats.json`
   - `build/continuity-report.md`
   - `build/project-report.md`
7. Show one playbook in `playbooks/`.
8. Close by pointing to the extension points in `scripts/`, `playbooks/`, and `templates/`.

## What To Emphasize

- chapter-scoped workflows
- diffable source files
- grounded agent collaboration
- packaging and review readiness
