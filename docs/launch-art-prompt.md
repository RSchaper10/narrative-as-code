# Launch Art Prompt

If you want promotional artwork instead of a screenshot, this prompt is designed for a DALL-E-style image that supports the repo launch without implying the tool is a full novel-writing autopilot.

## Primary Prompt

Create a cinematic editorial illustration for a project called **Narrative-as-Code Starter**.

Scene: a serious contemporary author-storyteller workspace near a stormy coastal window at dawn. On the desk are printed manuscript pages, handwritten notes, index cards, a fountain pen, a laptop with clean Markdown text, and a subtle diagram connecting chapters, canon, metadata, research, audio, and review. The mood should feel intelligent, grounded, and human-led, not futuristic hype. Suggest structure, craft, and coherence rather than magic automation.

Style: premium book-jacket-meets-modern-editorial-illustration, restrained color palette, Atlantic blues, weathered paper whites, graphite blacks, muted brass accents, soft atmospheric light, rich texture, tasteful composition, no cheesy sci-fi UI, no robots, no glowing holograms.

Composition: enough negative space for LinkedIn or GitHub social cropping. Make it readable at thumbnail size. Emphasize authorship, systems thinking, and storytelling craft.

## Variants

### More Technical

Add a wall pinboard or notebook spread showing chapter cards, canon notes, and build outputs connected by fine diagram lines, while keeping the scene grounded and analog-first.

### More Literary

Lean further into the writer’s desk, coastal weather, manuscript pages, and tactile objects. Keep any software elements subtle and secondary.

### More Brand-Neutral

Avoid visible product names on screens or papers. Focus on the emotional idea of structured storytelling.

## Negative Prompt Guidance

Avoid:

- robots
- neon cyberpunk interfaces
- generic AI head silhouettes
- fantasy spellbook aesthetics
- futuristic holograms
- stock-photo corporate teamwork vibes
