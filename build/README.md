# Build Artifacts

This directory is generated by the local build workflow.

Primary artifacts:

- compiled manuscript Markdown
- optional EPUB
- optional print-source DOCX
- `manuscript-stats.json`
- `continuity-report.json`
- `continuity-report.md`
- `project-report.json`
- `project-report.md`

These files are useful for review, handoff, and future automation.
