# One-Page Pitch

`Signal Harbor` is a coastal mystery about an archivist and a salvage diver who uncover a hidden municipal network built to monitor dangerous underwater signal events. The project is designed as a tiny, legible demonstration of a repository-native fiction workflow.

The story emphasis is not scale. It is workflow clarity:

- prose lives in chapters
- durable truth lives in canon
- project state lives in metadata
- support material lives outside the manuscript
