# Playbooks

These playbooks turn the starter into an operator-friendly workflow instead of a repo that depends on memory.

Use them when collaborating with agents, editors, or teammates who need a crisp, chapter-scoped task.

Included workflows:

- `review-chapter.md`
- `refine-chapter.md`
- `check-continuity.md`
- `update-metadata.md`
- `build-editor-packet.md`
