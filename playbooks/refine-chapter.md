# Playbook: Refine Chapter

## Goal

Improve one approved chapter against explicit constraints while preserving intent and canon.

## Inputs

- target chapter
- current chapter metadata
- any style/tone guardrails from `canon/story-bible.md`
- open questions that should remain unresolved

## Operator Prompt

1. State the refinement target clearly: clarity, tension, pacing, or atmosphere.
2. Preserve established facts, names, and unresolved mysteries.
3. Keep revisions chapter-scoped.
4. Surface any canon conflicts instead of silently resolving them.
5. Return a revised passage or full chapter only if requested.

## Expected Output

- summary of what changed
- updated prose or patch
- note on whether metadata summary should change
